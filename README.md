**电商政策与商品智能分析助手（自用）**  
基于 RAG（检索增强生成）与工具调用（Tool Use）的智能问答系统，专为电信行业政策文件分析与商品数据洞察设计。系统通过向量检索与自定义工具的结合，实现政策解析、商品相似度匹配、新品牌识别等复杂分析任务。

**核心功能**  
政策文件问答：上传政策文件（PDF/Word/TXT），通过自然语言提问，系统自动检索相关条款并生成精准回答。

商品相似度计算：输入商品名称或描述，系统基于向量相似度在全量商品表中查找相似商品，支持阈值筛选。

新品牌识别：对比已有商品库，自动识别新增品牌或商品，标记潜在竞争商品。

政策追踪分析：根据政策文件变化，推荐可能受影响的商品类别，辅助业务决策。

工具集成：将上述分析逻辑封装为 LangChain Tools，Agent 可自主组合调用，实现复杂业务流程。
